# 🌐 Simple Portfolio — Rahif Khan

A clean and responsive personal portfolio website built with HTML and CSS.

## 🔗 Live Preview
[View Portfolio](https://rahifkhan22.github.io/Simple-Portfolio)

---

## 📋 Sections

- **Hero** – Introduction and resume download button
- **About** – Brief personal and academic background
- **Skills** – Technologies and languages known
- **Projects** – Highlighted personal/academic projects
- **Certificates** – Industry job simulations and certifications
- **Education** – Academic history
- **Languages** – Spoken language proficiency
- **Contact** – Phone, email, LinkedIn, and GitHub

---

## 🛠️ Built With

- HTML5
- CSS3
- Google Fonts (Poppins)

---

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/rahifkhan22/Simple-Portfolio.git
   ```
2. Open `index.html` in your browser

No installations or dependencies required.
